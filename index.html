Project - Video To Text AI Assisted

The code 1st converts a video into audio file and then it converts the audio into 30 chunks which are then converted to text individually using recognize_google then the chunks are sent to OpenAI "model="gpt-3.5-turbo-instruct" which gives us the results.

Project Results

STEP 1

The code 1st converts a video into an audio file. Here is an example video used.

STEP 2

The code then converts audio into text by splitting the audio into chunks of 30 words each using the recognize_google tag.

Then it uses OpenAI API to send a prompt along with the complete extracted text compiled from the chunks, which is returned back in the form of an output.
